# RO 物價追蹤

一個跑在**你自己電腦**上的小程式：用你的 gnjoy 帳號登入
[RO 物價查詢](https://event.gnjoy.com.tw/Ro/RoShopSearch)，
釘選你想追蹤的商品，定時自動幫你查最新價格，打開網頁就能看到。

> 為什麼要跑在自己電腦？因為那個物價查詢網站需要登入，而且只有你的電腦
> （你的網路）連得到。程式登入用的帳密只會存在你電腦本機的 `.env` 檔，
> **不會**上傳到任何地方。

---

## 怎麼用（Windows）

1. 安裝 [Node.js](https://nodejs.org)（下載 LTS 版，一路下一步即可）。
2. 下載這個 `RoPriceTracker` 資料夾到你電腦。
3. 點兩下 **`start.bat`**。
   - 第一次會自動安裝需要的東西（含瀏覽器，需幾分鐘）。
   - 接著會跳出 `.env` 記事本，請填入你的帳號密碼後存檔：
     ```
     RO_ACCOUNT=你的帳號
     RO_PASSWORD=你的密碼
     ```
   - 存檔後再點一次 `start.bat`。
4. 瀏覽器會自動打開 <http://localhost:5178>。輸入商品名稱就能釘選追蹤。

之後每天要看物價，點 `start.bat`、打開網頁就好。

## 在手機上使用 📱

電腦負責登入、查價、運算；手機只是打開網頁看結果。

1. 讓 `start.bat` 在電腦上保持執行中。
2. 啟動時視窗會印出「手機打開」的網址，長得像 `http://192.168.x.x:5178`。
3. **手機連上跟電腦同一個 Wi-Fi**，在手機瀏覽器輸入那個網址。
4. 想要像 App 一樣：
   - iPhone（Safari）：分享鍵 → 「加入主畫面」。
   - Android（Chrome）：右上選單 → 「加到主畫面／安裝應用程式」。
   之後點主畫面的 🪙 圖示就會全螢幕打開，跟 App 一樣。

> **連不上？** 第一次啟動時 Windows 會跳防火牆提示，請勾「私人網路」並按
> 「允許存取」。若當時按掉了，到「Windows 防火牆 → 允許應用程式」把 Node.js
> 的私人網路打勾即可。手機和電腦也要在同一個 Wi-Fi。
>
## 在不同網路下使用（人在外面 · Tailscale）🌐

想在外面用手機網路也能開？用 **Tailscale**：它把你自己的裝置私下連在一起，
**不會**把 App 公開到網路上，只有你自己連得到，所以很安全、也免費。

1. **電腦**：到 <https://tailscale.com/download> 下載 Windows 版，安裝後用一個
   帳號登入（Google／Microsoft 皆可）。
2. **手機**：App Store／Play 商店搜「Tailscale」安裝，用**同一個帳號**登入，
   並把它打開（保持連線）。
3. 讓電腦上的 `start.bat` 保持執行。啟動視窗會多出一個
   **「不同網路也能用 · Tailscale」** 的網址，長得像 `http://100.x.x.x:5178`。
   - 找不到的話，在電腦命令列打 `tailscale ip -4` 也能看到這個 100 開頭的 IP。
4. 手機（不管用 Wi-Fi 還是行動網路）瀏覽器輸入那個 `100.x.x.x:5178` 就能用，
   一樣可以「加到主畫面」。

> 只要電腦有開機、`start.bat` 有在跑、兩邊 Tailscale 都登入同一帳號，
> 你在任何網路下都能連。電腦關機就連不到（因為運算在電腦上）。

## 怎麼用（Mac / 手動）

```bash
cd RoPriceTracker
npm install
cp .env.example .env      # 然後編輯 .env 填入帳密
npm start                 # 打開 http://localhost:5178
```

---

## 功能

- **釘選商品**：輸入商品名稱即可加入追蹤，會立刻查一次。
- **定時自動更新**：預設每 30 分鐘更新一次（可在網頁上調整）。
- **打開就看到**：每個商品顯示最新價格、最低價、更新時間。
- **手動更新**：單一商品或「全部立即更新」。

## 爬蟲還沒對準？跑「擷取模式」給我

因為做的時候還沒看過你登入後的實際頁面，抓價格的欄位是先用猜的。
如果查出來的東西怪怪的（例如顯示原始表格、或查不到），
請在資料夾裡開命令列跑：

```bash
npm run capture -- 商品名稱
```

例如 `npm run capture -- 藍色寶石`。跑完會產生 `capture.json`（和截圖
`capture.png`），把 **capture.json** 貼給我，我就能把欄位對準。

> `.env`、`data/`、`capture.*` 都已被 `.gitignore` 擋掉，不會上傳。

## 設定（.env）

| 設定 | 說明 |
| --- | --- |
| `RO_ACCOUNT` / `RO_PASSWORD` | 你的 gnjoy 帳密（只存在本機） |
| `RO_SEARCH_URL` | 物價查詢頁，通常不用改 |
| `RO_LOGIN_URL` | 登入頁；留空則在查詢頁上直接找登入欄位 |
| `PORT` | 本機網頁埠號，預設 `5178` |
| `HEADLESS` | `true`=背景執行；除錯想看瀏覽器畫面設 `false` |

---

## 安全提醒

- 請勿把 `.env` 或 `capture.*` 貼到公開的地方（裡面有你的帳密／查詢內容）。
- 如果你的密碼曾經在聊天／截圖中外流，請儘快到 gnjoy 更換密碼。
