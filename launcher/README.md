# 一鍵啟動器（One-Click Launcher）

雙擊一次，就把你常用的程式、網址、檔案全部打開，不用每次重開機再一個一個點。

## 怎麼用（3 步驟）

1. 用記事本打開 **`apps.txt`**。
2. 把你要開的東西一行一個寫進去（網址、程式路徑、檔案、資料夾都可以），存檔。
3. 雙擊 **`啟動.bat`** —— 全部一起打開。

就這樣。之後要多開或少開什麼，只要改 `apps.txt`，不用動到別的檔案。

## `apps.txt` 寫法

一行一個項目，例如：

```
https://www.google.com
notepad
C:\Program Files\Google\Chrome\Application\chrome.exe
D:\我的資料\報表.xlsx
```

- 行首是 `#` 的整行是**註解**，不會執行 —— 可以拿來備註，或暫時把某一項關掉（前面加 `#` 即可）。
- 空白行會被略過。
- 路徑**不用加引號**，有空格也沒問題。

### 怎麼找到程式的路徑？

在開始功能表找到那個程式 → 右鍵 → **開啟檔案位置** → 對捷徑再右鍵 → **內容** → 「目標」欄位那串就是路徑，複製貼到 `apps.txt`。

## 想要「開機就自動啟動」？（可選）

目前是雙擊執行。如果之後想開機自動跑：

1. 按 `Win + R`，輸入 `shell:startup`，按 Enter（會打開「啟動」資料夾）。
2. 對 `啟動.bat` 按右鍵 →「建立捷徑」，把捷徑丟進剛剛那個「啟動」資料夾。

這樣每次開機登入後就會自動執行。想取消就把那個捷徑刪掉。

## 小提醒

- 每個項目之間預設會停 1 秒，避免一次開太多卡住。想更快可以打開 `啟動.bat`，把 `timeout /t 1` 那行的數字改小或刪掉。
- 網址若含 `!` 這種特殊符號可能會失效（很少見），一般網址都沒問題。
